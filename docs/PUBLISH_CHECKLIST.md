# Publish Checklist

## Metadata

- [ ] Repository name and description updated.
- [ ] `LICENSE` present.
- [ ] `README.md` reflects current experiment structure.
- [ ] `CITATION.cff` reviewed (authors, version, optional repository URL).

## Repository hygiene

- [ ] `artifacts/` and `logs/` ignored.
- [ ] No private credentials or machine-specific paths in tracked files.
- [ ] No large model weights committed by mistake.

## Reproducibility

- [ ] Step statuses updated in `experiments/ROADMAP.md`.
- [ ] Latest result notes match actual outputs.
- [ ] Run commands for key experiments are documented.

## Community defaults

- [ ] `CONTRIBUTING.md` present.
- [ ] `CODE_OF_CONDUCT.md` present.
- [ ] `SECURITY.md` present.
- [ ] `CHANGELOG.md` updated.
